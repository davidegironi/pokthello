About
===
pOkTHELLO is the web page front-end that calls the api to fetch pokemon Shakespeare descriptions.

## Install
```bash
npm install
```

## Run
```bash
npm start
```

## Build
```bash
npm run build
```

## Linter
```bash
npm run lint
```
